Govstrap.io quickstart guide


Case Scenario 1


1. Add Govstrap

We have based our styling on the latest version of Bootstrap 4 . Because we expect this beta version to suffer only minor modification until the final release, we have included it in our styling.

If you do feel adventurous enough you can fork the project

Download the Govstrap CSS file from our CDN.

Load it in your HTML head section like this:


Example


<!-- Govstrap styling -->
<link href="govstrap.css" rel="stylesheet">

2. You're done!

You can now use all of the Govstrap classes and elements in your site.
